Introdução ao HTML

O que é HTML?

O HTML é uma linguagem usada para criar a estrutura das páginas Web. É uma linguagem de marcação, serve para organizar e descrever o conteúdo de um site.

Com o HTML, você define quais serão os Títulos, parágrafos, imagens, links, listas, formulários, botões e tabelas em uma página web.

Pra que serve o HTML?

O HTML serve para dizer ao navegador o que é cada coisa dentro da página. Sem o HTML, o navegador não saberia como organizar ou mostrar as informações. Ele não saberia o que é um título, um link, um parágrafo e etc.

Principais Tags de Formatação:

Essas são as principais Tags de Formatação em HTML, usadas para alterar a aparência e significado do texto:

<b>

<strong>

<i>

<em>

<mark>

<small>

<del>

<ins>

<sub>

<sup>

Exemplos de uso de algumas tags:

A tag <b> serve para destacar o texto visualmente para chamar a atenção, sem atribuir uma importância extra ao conteúdo. É apenas um negrito.

Texto em Negrito

A tag <strong> é usada quando você quer indicar que aquele trecho de texto é muito importante. Os navegadores exibem em negrito, mas para um leitor de tela, essa tag faz com que a voz do software mude o tom para dar ênfase.

Texto importante

A tag <i> é usada para termos técnicos, palavras em outros idiomas, pensamentos de personagens ou nomes de obras. Ela inclina o texto sem necessariamente dar uma "ênfase de voz".

Texto em Itálico

A tag <em> é usada para dar ênfase ao que está sendo dito, como se você estivesse mudando a entonação da voz ao ler. Tem valor semântico.

Texto em ênfase

A tag <mark> funciona como um marca-texto amarelo. É útil para destacar termos que o usuário pesquisou em uma página ou partes de um texto que são relevantes em um contexto específico.

Texto marcado

A tag <small> indica "letras miúdas", como avisos de direitos autorais (Copyright), termos de licenciamento ou notas de rodapé.

Texto secundário

A tag <del> representa um texto que foi removido do documento. É muito comum em listas de tarefas ou para mostrar preços antigos em um e-commerce

Texto deletado

A tag <sub> deixa o texto levemente abaixo da linha principal e em tamanho menor. Seu uso principal é em fórmulas químicas ou em notações matemáticas.

H20

A tag <sup> deixa o texto levemente acima da linha principal. Muito usado para potências matemáticas, notas de rodapé indicadas por números ou abreviações.
